# 麒麟软件生态适配管理平台

## 简介

麒麟软件生态适配管理平台(KAIM, KylinEco Adaption Information Management)是基于[Laravel](https://laravel.com)及[Dcat Admin](http://www.dcatadmin.com/)开发的用于管理银河麒麟的软硬件适配数据及适配需求的的一体化平台
